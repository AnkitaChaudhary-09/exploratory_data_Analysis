# exploratory_data_Analysis
# 🪔 Diwali Sales Analysis

## About the Project

This project is based on a Diwali sales dataset. The goal was to understand customer buying behavior during the festive season and find useful insights that can help businesses make better decisions.

Using Python and data analysis libraries, I cleaned the data, explored different customer groups, and visualized sales patterns to understand who buys more, what products are popular, and which customer segments contribute the most to sales.

---

## Problem Statement

Businesses collect a large amount of sales data, but raw data alone does not provide useful information.

The challenge was to analyze the sales data and answer questions such as:

* Who are the most valuable customers?
* Which age group spends the most?
* Which states generate the highest sales?
* Which product categories are most popular?
* How can businesses use this information to improve sales?

---

## What I Did

### Data Cleaning

* Loaded the dataset using Pandas
* Checked for missing values
* Removed unnecessary columns
* Prepared the data for analysis

### Exploratory Data Analysis (EDA)

I analyzed sales based on:

* Gender
* Age Group
* Marital Status
* State
* Occupation
* Product Categories

### Data Visualization

Using Matplotlib and Seaborn, I created charts to better understand customer behavior and sales trends.

---

## Key Insights

Some important findings from the analysis:

* Female customers contributed a large portion of total sales.
* Customers aged 26–35 were among the most active buyers.
* Married customers showed higher purchasing behavior.
* Certain states generated significantly higher revenue.
* Some product categories were consistently more popular than others.

## How This Helps a Business

The insights from this project can help businesses:

* Target the right customers with marketing campaigns.
* Focus on customer groups that generate the most revenue.
* Improve inventory planning based on popular products.
* Make better business decisions using data instead of assumptions.
